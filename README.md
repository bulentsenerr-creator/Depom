# Kitap Kütüphanem — v3.2 (Düzeltilmiş UI + Çalışan Butonlar)

Bu paket, senin bildirdiğin sorunları düzelterek hazırlanmıştır:

## Düzeltmeler
- Barkod Oku / Bilgileri Getir / Alt menü butonları ve tüm sheet (Detay/Filtre) butonları **çalışır**.
- Alt menüde Filtre butonu **Filtre sheet'i açar** (gerekirse Raflar sayfasına geçirir).
- UI yeniden tasarlandı: daha temiz, daha okunaklı, tutarlı.

## Özellikler
- Barkod okuduktan sonra otomatik bilgi getirme
- Dinamik raflar: raf adını yaz → raf otomatik oluşur
- Gerçek zamanlı filtreleme + sıralama
- Manuel kapak yükleme

## Çalıştırma
```bash
python -m http.server 8080
```
